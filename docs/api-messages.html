Messages

Envoyer un message à un agent et récupérer la réponse structurée.

POST /v1/messages

Requête
{
  "agentId": "agt_123",
  "sessionId": "sess_abc",
  "content": "Bonjour"
}
Réponse
{
  "id": "msg_001",
  "sessionId": "sess_abc",
  "status": "completed",
  "answer": "Bonjour, comment puis‑je aider ?"
}
