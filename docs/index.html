Bienvenue

Cette documentation présente comment intégrer un agent Findora: clés API, endpoints, webhooks et bonnes pratiques.

Ce que vous pouvez faire

  • • Envoyer des messages à un agent et lire les réponses
  • • Ajouter des fichiers pour enrichir le contexte
  • • Suivre les sessions et recevoir des webhooks d’événements

Messages

Envoyer un message à un agent et récupérer sa réponse (POST /v1/messages).

Exemple cURL
curl -X POST https://api.findora.ai/v1/messages \
-H "Authorization: Bearer $FINDORA_API_KEY" \
-H "Content-Type: application/json" \
-d '{"agent_id":"agent_123","input":{"type":"text","text":"Bonjour !"}}'

Fichiers

Uploader un fichier (POST /v1/files).

Exemple cURL
curl -X POST https://api.findora.ai/v1/files \
-H "Authorization: Bearer $FINDORA_API_KEY" \
-F "file=@/chemin/vers/document.pdf" \
-F "purpose=knowledge"

Sessions

Lister et clore des sessions (GET /v1/sessions, POST /v1/sessions/close).

Lister (cURL)
curl -H "Authorization: Bearer $FINDORA_API_KEY" https://api.findora.ai/v1/sessions
